# Project Images

Put your project images in this folder.

## Supported formats:
- JPG/JPEG
- PNG
- WebP (recommended for web)
- SVG

## Recommended naming:
- `pln-robot.jpg` - Smart Robot System
- `kri-robot.jpg` - Autonomous Navigation Robot
- `jats-editor.jpg` - JATS Editor Platform
- `ai-document.jpg` - AI Document Processing
- `fuzzy-logic.jpg` - Fuzzy Logic Production System
- `government-monitoring.jpg` - Government Monitoring System

## Image guidelines:
- Resolution: 1200x675px (16:9 ratio) or similar
- File size: Keep under 500KB for fast loading
- Use descriptive filenames (lowercase with hyphens)

## Usage in code:
Images in this folder are referenced as `/projects/filename.jpg`

Example:
```javascript
{
  title: "Smart Robot System",
  image: "/projects/pln-robot.jpg",
  // ...
}
```
